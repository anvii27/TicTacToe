# TicTacToe Game in Java

This is a simple TicTacToe game implemented in Java using Swing for the graphical user interface.

## Features
- Two-player game (X and O).
- Graphical interface with buttons representing the board tiles.
- Detects wins (horizontal, vertical, diagonal) and ties.
- Highlights the winning line when a player wins.
- Displays turn information and game result.
